What is Mastermind?


Mastermind is a code-breaking game, originally for two players but replicated here as a singleplayer game against an auto-generated code. You can read about it here, but to explain it briefly:

A secret code, consisting of 4 (not necessarily distinct) letters from A to F, will be generated. Your job is to crack it, by making and modifying guesses. After each guess, it will be indicated how "close" your guess is, as explained below. You have 12 guesses. Good luck!

To make a guess, type exactly one capital letter from the set {A,B,C,D,E,F} into each box. After pressing "Enter", you will be shown four icons in no particular order: each ✖ represents a letter which isn't in the code, each ✔ represents a letter which is in the code but in the wrong position, and each ✅ represents a letter which is in the code and in the right position.
